# Intent: Convert a string to upper case (or lower case)

```ring
Q("hello world").Uppercased()
#--> "HELLO WORLD"
```

Lower case is the mirror:

```ring
Q("HELLO").Lowercased()
#--> "hello"
```

- **Methods:** stzString.Uppercased, stzString.Lowercased
- **Tags:** uppercase, lowercase, capitalize, upper, lower, case, capitals, make text uppercase, all caps
- **See also:** trim
